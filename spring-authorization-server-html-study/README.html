Spring Authorization Server 1.5.8 中文学习小书

一个面向 conference 微服务项目落地的静态 HTML 学习项目。

版本锁定： 本项目内容固定基于 Spring Authorization Server 1.5.8 官方文档。每章正式编写前，需要先核对对应官方页面。

当前进度

  • 已完成：首页、基础样式、基础脚本。
  • 已完成正式草稿：第 0 章、第 1 章、第 2 章。
  • 已创建占位页：第 3～11 章，避免目录链接 404。

项目约定

官方内容与 conference 项目落地建议必须分开写。凡是架构拆分、表结构、Gateway Header、Feign Token 策略等项目实现方案，都要明确标注“这是项目落地建议，不是官方原文”。
